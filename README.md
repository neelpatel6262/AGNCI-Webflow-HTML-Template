# 🌐 AGNCI – Webflow HTML Website Template

A modern agency-style website template, meticulously recreated from Webflow using pure HTML5, CSS3, and responsive design principles. Built from scratch without frameworks—a perfect learning project and professional template.

## 🎨 Preview

Watch the live website in action on [Vercel](https://agnci-webflow-html-template.vercel.app/)

## 📖 Project Story

I recently started learning HTML, CSS, and responsive design properties. To practice and strengthen my skills, I picked up the AGNCI Webflow template and recreated it using pure HTML, CSS, and responsive principles.

Instead of exporting from Webflow, I used the **VisBug extension** to inspect elements and rebuild the entire layout from scratch. This hands-on approach helped me develop a deeper understanding of clean code structure, responsive design patterns, and professional web architecture.


## 🛠 Technologies

- **HTML5** - Semantic markup and accessibility
- **CSS3** - Advanced styling, flexbox, grid, and media queries
- **Responsive Design** - Mobile-first approach with comprehensive breakpoints
- **Vercel** - Deployment and hosting
- **VisBug** - Element inspection and design analysis tool

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/neelpatel6262/AGNCI-Webflow-HTML-Template
cd AGNCI-Webflow-HTML-Template
```

### Open Locally

Simply open `index.html` in your preferred browser. No build process or dependencies needed!

```bash
# On macOS
open index.html

# On Windows
start index.html

# On Linux
xdg-open index.html
```

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Import your repository
4. Deploy with one click!
