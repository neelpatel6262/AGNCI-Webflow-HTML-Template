# 🌐 AGNCI – Webflow HTML Website Template

A modern agency-style website template, meticulously recreated from Webflow using pure HTML5, CSS3, and fully responsive design. Built from scratch without frameworks—a perfect learning project and professional template.

---

## 📖 Project Description

I recently started learning HTML, CSS, and responsive design properties. To practice and strengthen my skills, I picked up the AGNCI Webflow template and recreated it using pure HTML, CSS, and responsive principles.

Instead of exporting from Webflow, I used the **VisBug extension** to inspect elements and rebuild the entire layout from scratch. This hands-on approach helped me develop a deeper understanding of clean code structure, responsive design patterns, and professional web architecture.

**Live Demo**: [AGNCI Webflow HTML Template](https://agnci-webflow-html-template.vercel.app/)

## 🔄 The Process

**Design Analysis**
- Used VisBug extension to inspect the original Webflow template
- Analyzed layout structure, spacing, typography, and color schemes
- Documented all design elements and components

**HTML Structure**
- Created semantic HTML5 markup with proper heading hierarchy
- Implemented accessible forms and navigation
- Organized code into logical sections

**CSS Development**
- Built comprehensive CSS with variables for maintainability
- Implemented mobile-first responsive design approach
- Created 5 media query breakpoints for all device sizes
- Added dark mode and accessibility features

**JavaScript Implementation**
- Added mobile menu toggle functionality
- Implemented smooth interactions and form validation
- Ensured cross-browser compatibility

**Testing & Optimization**
- Tested across multiple devices and browsers
- Optimized for performance and accessibility
- Verified responsive design across all breakpoints
